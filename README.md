# OneTab

A web application that lets users store, organize, and manage files in a structured folder system — all in one place.

---

##  Features

*  Secure user authentication (signup & login)
*  Create and manage folders
*  Upload and store files per user
*  Organized file structure for easy access
*  Edit and delete files and folders
*  File preview support

---

##  Tech Stack

* **Backend:** Flask
* **Database:** SQLite
* **ORM:** SQLAlchemy
* **Frontend:** HTML, CSS (you can add JS if you used it)

---

##  Overview

OneTab is designed as a lightweight file management system similar to cloud storage platforms.
Each user gets their own private space to upload, organize, and manage files efficiently.
